# Projects Repository

This repository contains a comprehensive collection of web development projects created for learning and studying various modern web technologies and frameworks. The primary focus is on mastering Next.js, Auth.js, Tailwind CSS, and modern build tools like Turborepo and Vite. Each project serves as a practical learning exercise and demonstration of different concepts, patterns, and best practices in web development.

## Primary Learning Focus

The main technologies and tools being studied in this repository:

### Core Technologies

- **Next.js** (v15): Server-side rendering, API routes, and full-stack development
- **React** (v19): Component-based UI development with latest features
- **Auth.js**: Authentication and authorization implementation
- **Tailwind CSS** (v3.4.17): Utility-first CSS framework and styling patterns
- **TypeScript** (v5.x): Type safety and modern JavaScript features

### Build Tools & Development Environment

- **Turborepo** (v2.4.2): Monorepo management and build optimization
- **Vite** (v6.x): Modern frontend build tooling
- **pnpm** (v10.4.1): Fast, disk space efficient package manager
- **Bun** (v1.2.12): Alternative JavaScript runtime

### UI & Component Libraries

- **Shadcn UI**: Accessible UI component system
- **Radix UI**: Headless UI primitives
- **Bootstrap** (v5.3.5): CSS framework for responsive design

### Additional Technologies

- **SQLite** (better-sqlite3 v9.x): Database management and integration
- **Cloudinary**: Media handling and optimization
- **TanStack Query**: Data fetching and caching
- **Zustand**: Lightweight state management
- **React Hook Form + Zod**: Form handling and validation

## Learning Objectives

These projects are designed to:

- Master Next.js 15 framework and its ecosystem
- Implement secure authentication with Auth.js
- Create responsive designs with Tailwind CSS
- Optimize build processes with modern tools
- Explore monorepo architecture with Turborepo
- Practice modern React 19 patterns and features
- Learn TypeScript best practices
- Build scalable component libraries
- Implement data fetching and state management
- Explore and understand different web development frameworks and tools
- Practice implementing various features and patterns
- Learn best practices for different technologies
- Build a portfolio of practical examples
- Demonstrate the evolution of web development skills

## Common Development Scripts

Most projects support the following common scripts:

- `npm run dev` / `pnpm dev` / `bun dev`: Start development server
- `npm run build` / `pnpm build` / `bun run build`: Build for production
- `npm start` / `pnpm start`: Start production server
- `npm run lint` / `pnpm lint`: Run linting
- `npm run type-check` / `pnpm typecheck`: Run TypeScript checks
